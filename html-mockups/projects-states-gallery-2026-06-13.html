Let's start with a project.

A project is your workspace on Toko — everything you make lives inside one. Collections, generators, vendors, media, team access, royalties.

01

One project, many collections

Most studios run several themed drops under one project — spring, summer, special editions. Team, beneficiaries, and whitelists are shared across them.

02

Bring your team in early

Add collaborators by principal. Roles like Super Admin, Treasurer, Admin, and Maintainer decide what each person can do.

03

You can ship without code

Define a collection, set up a generator or upload tokens, configure a vendor, and you're live on the Launchpad — no contracts to deploy.

What's inside a project
Collections + Generators + Vendors + Media + Team + Beneficiaries
