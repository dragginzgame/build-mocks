Collection archetypes — Phase 1 UI changes.

Every front-end change Phase 1 of the archetype system requires, per collection-archetypes.md (the Phase 1 spec, ratified 2026-07-20). Three archetypes only (Curated / Generated / Editioned Art), chosen freely from cards. No wizard, no "Soon" cards, no locked capabilities, no upgrade paths, no collection-level badge — all growth presentation belongs to the Phase 3 Project Mastery tab and is absent here. Supersedes frames 01–02 of collection-creation-states-gallery-2026-06-29.html (seven cards with "Soon" states) and the "Unlock more capacity — coming soon" note in supply-states-gallery-2026-06-29.html. The Rarity page needs no Phase 1 change: the existing rarity gallery's Weighted Setup already shows the three available tiers with a plain "more can be unlocked later" access note — nothing further (and no unlock requirements) should be added there. Go-live readiness surfaces on the Draft overview's Collection status checklist (frame 06), not in a separate modal.

08Guards page · seeded defaults + contradiction warningThe archetype seeds the duplicates guard from its edition mode (Generated → off; Curated / Editioned → on).
