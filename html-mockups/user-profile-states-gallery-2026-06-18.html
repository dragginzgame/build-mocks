User profile — states & stages.

The /profile/:principal page — one base profile shared by collectors and creators. Public view is what any visitor sees; Owner view adds private sections (copyable IDs, shipping addresses) that only the account holder can see. Grouped below: Public, Owner, system states, then the modals — edit profile and add / edit / remove address.

What any visitor sees. No edit affordances, no copyable IDs, no private sections. Empty optional fields (about, website, socials) simply don't render.
