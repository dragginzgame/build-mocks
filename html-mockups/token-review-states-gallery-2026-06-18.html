Toko · Consistency reference

Tokens — Review.

The gate between Draft and Live. Review tokens are locked for content (editing happens in Draft); here you screen content, set them Live, or send them back to Draft. Single and bulk.

Lifecycle: Add · Draft · Review · Live · Minted
01Single — Review (stage-only)A Review token is stage-only — the backend rejects every non-stage update. Nothing on the definition (name, description, tags, attributes, rarity, composition, assets, supply) can change. The only actions are setting it Live or returning it to Draft to edit.

07Set Live blocked — definition capThe live-definition cap is a hard gate at Review → Live (unlike copy supply, which may be oversubscribed). Two cases: the cap is already full (nothing can go Live), or there's partial room (only some of the selection fits). Draft and Review are never capped — only the move to Live.
