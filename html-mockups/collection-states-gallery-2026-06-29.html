This permanently deletes the collection. This can't be undone.

Delete is only available pre-Live (Draft / Review). A Live collection can't be deleted — there's no delete affordance in its Danger zone — so a "blocked" modal is never reached.
07Collection overview image · cycle check (balance OK)Saving a collection banner or thumbnail stores it in blob storage, funded from project cycles. The guard is the same for either image — the modal just names the one you're adding (thumbnail shown here, banner in 08–09). Before it saves, Toko estimates the cost and checks the balance — the same guard used for every upload (see the Cycles & uploads gallery). With headroom, it saves normally. On a Live collection these media edits save directly.
