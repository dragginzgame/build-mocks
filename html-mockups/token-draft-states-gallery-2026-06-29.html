Tokens — Draft.

Preparing token definitions in Draft: manage the list, edit a single token (tabbed) or many in bulk, and move complete tokens to Review. Editing — composition, attributes, rarity, supply — is only possible while a token is in Draft.

11Single edit — Supply · copies allowedWhen the collection allows copies: choose Single / Capped (Unlimited gated, not yet available), set whether copies stack, and set a desired issue number. The issue number is one value shared by every copy; it's validated immediately but only held once the token reaches Review. Capped shows the per-token hard maximum from Supply; numbered editions are always Individual.
